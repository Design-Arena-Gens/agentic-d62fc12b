# Sci‑Fi Chase Scene

A Next.js app rendering a 20-second cinematic spacecraft chase using Three.js with React Three Fiber, deployable to Vercel.

## Scripts

- `npm run dev` – start dev server
- `npm run build` – build for production
- `npm start` – start production server

## Deploy

```
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-d62fc12b
```
